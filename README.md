# SQL Data Cleaning Project – Layoffs Dataset

## Project Overview
This project focuses on cleaning and preparing a raw layoffs dataset using SQL to make it more accurate, consistent, and analysis-ready.

The primary objective was to transform messy real-world data into a structured dataset by handling:
- Duplicate records
- Standardization issues
- Null / blank values
- Inconsistent formatting
- Unnecessary columns

This project demonstrates practical SQL skills commonly used in data analytics, business intelligence, and pre-analysis data preparation.

---

# Dataset
The dataset contains company layoff information, including:
- Company Name
- Location
- Industry
- Total Laid Off
- Percentage Laid Off
- Date
- Stage
- Country
- Funds Raised (Millions)

---

# Project Goals
## Key cleaning objectives:
- Create a staging table to preserve raw data
- Identify and remove duplicate rows
- Standardize inconsistent text values
- Fix formatting issues (dates, spaces, country names)
- Handle null and blank values
- Remove unnecessary columns after cleaning
